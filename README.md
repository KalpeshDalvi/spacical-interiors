# Spacical Interiors – Next.js Site

This is a marketing website for **Spacical Interiors**, built with Next.js 14 (App Router) and Tailwind CSS.

## Getting started

1. Install dependencies:

```bash
npm install
# or
yarn
```

2. Run the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm start
```

## Images

Hero and project images reference files under `/public/images` such as:

- `/images/hero-living-room.jpg`
- `/images/hero-detail-1.jpg` … `hero-detail-4.jpg`
- `/images/about-1.jpg` … `about-4.jpg`

Add your own interior photos with those file names into the `public/images` folder.

## Contact form

The contact form on `/contact` is front-end only. Connect it to a service like Formspree/Getform or a custom API endpoint before going live.
